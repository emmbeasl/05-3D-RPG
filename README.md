# 05-3D-RPG
Project 5 3D RPG

# Implementation 
A 3D game that includes a third-person character which moves with keyboard and mouse control
Camera follows players movement
Interactive objects
Non-player agents
Dialogue-based conversation with one agent
One hand-held item
A scoring system
